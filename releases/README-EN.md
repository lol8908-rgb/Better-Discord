# Better Discord v1.0.0 - English 🇬🇧

## Installation

### Automatic (Windows):
Double-click `install-EN.bat`

### Automatic (Linux/Mac):
```bash
chmod +x install-EN.sh
./install-EN.sh
```

### Manual:
1. Download `BetterDiscord.plugin.js`
2. Save it in:
   - **Windows**: `%APPDATA%/BetterDiscord/plugins/`
   - **Linux**: `~/.config/BetterDiscord/plugins/`
   - **macOS**: `~/Library/Application Support/BetterDiscord/plugins/`

3. Restart Discord
4. Settings → BetterDiscord → Plugins
5. Enable "Better Discord"

## Features ✨

🖼️ Background images (JPG, PNG, GIF - animated!)
🎨 UI enhancements
🔤 Font customization
⚙️ Full customization

## Help

- Plugin not working? → Restart Discord
