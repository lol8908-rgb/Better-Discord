# Better Discord v1.0.0 - Deutsch 🇩🇪

## Installation

### Automatisch (Windows):
Doppelklick auf `install-DE.bat`

### Automatisch (Linux/Mac):
```bash
chmod +x install-DE.sh
./install-DE.sh
```

### Manuell:
1. Lade `BetterDiscord.plugin.js` herunter
2. Speichere es in:
   - **Windows**: `%APPDATA%/BetterDiscord/plugins/`
   - **Linux**: `~/.config/BetterDiscord/plugins/`
   - **macOS**: `~/Library/Application Support/BetterDiscord/plugins/`

3. Discord neustarten
4. Einstellungen → BetterDiscord → Plugins
5. "Better Discord" aktivieren

## Features ✨

🖼️ Hintergrundbilder (JPG, PNG, GIF - animiert!)
🎨 UI-Verbesserungen
🔤 Schriftanpassung
⚙️ Vollständige Anpassbarkeit

## Hilfe

- Plugin funktioniert nicht? → Discord neustarten
- Hintergrundbild wird nicht angezeigt? → URL überprüfen
